# DJ LAKHA Website

Welcome to your new luxury website!

## How to View
Simply double-click the `index.html` file to open it in your web browser.

## Customization Guide

### 1. Images
- [x] Background image in `styles.css` has been updated to use local asset `assets/portfolio_dance_floor.png`.
- [x] Portfolio images have been generated and added to `assets` folder. `index.html` and `media.html` have been updated.
- You can replace these images in the `assets` folder with your own photos (keep the filenames the same or update the code).

### 2. Honeybook Integration
- Go to your Honeybook account and get the "Contact Form Embed Code".
- Open `index.html` and replace the `<div class="honeybook-placeholder">...</div>` with your embed code.

### 3. Colors & Fonts
- You can change the primary Gold color in `styles.css` by editing the `--color-primary` variable at the top of the file.
