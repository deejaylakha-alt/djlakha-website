## 7. Technical Performance Checklist

### Core Web Vitals Targets:
| Metric | Target | What It Measures |
|---|---|---|
| LCP (Largest Contentful Paint) | < 2.5s | How fast the main content loads |
| FID (First Input Delay) | < 100ms | How quickly the site responds to clicks |
| CLS (Cumulative Layout Shift) | < 0.1 | Visual stability (no jumping elements) |

### Quick Wins:
- [ ] Compress all images to WebP format (50-70% smaller)
- [ ] Add `width` and `height` attributes to all `<img>` tags (prevents CLS)
- [ ] Move Lucide icons to local file instead of CDN (eliminates render-blocking)
- [ ] Add `<link rel="preload">` for hero image
- [ ] Minify CSS and JS for production
- [ ] Enable Gzip/Brotli compression on Hostinger

### Hosting:
- Hostinger has good CDN support — enable it
- Consider Cloudflare free tier for additional caching + CDN
- Ensure HTTPS is enabled (required for Google ranking)
